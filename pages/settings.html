Settings

Everything for appearance, sync, and backups in one clean place.

Appearance — controls how the whole app looks.

Dark / Light mode — toggle between dark (default) and light themes. The first time you try light mode, you'll get a friendly warning — click again within 60 seconds to confirm.

Background Image — set a full-screen background behind the panels. Choose from presets (forest, tavern, dungeon, etc.), upload your own image file, or paste a URL. Click Remove Background to go back to plain.

Accent Color — changes the highlight colour used for borders, buttons, headers, and active elements across the whole app. Pick any colour you like.

Text Size — use A− / A+ or the slider to scale all text globally. Range is 85%–140%. Useful for larger screens or accessibility.

Font Style — choose from 9 font options. All are built-in system fonts (except Roboto which loads from Google Fonts) — they work offline.

All appearance settings are saved to your browser and persist across sessions.

Account and Cloud Sync — sign in with Google to keep your characters backed up and synced across devices.

Sign In with Google — links your characters to your Google account. No password needed — uses Google's secure sign-in.

Once signed in, every change auto-syncs to the cloud within a few seconds. You can open the sheet on any device and your characters will be there.

Sync Up Now — manually push your current local data to the cloud. Use this if you've made changes offline and want to force an immediate upload.

Sync Down Now — pull the latest cloud data down to this device. Use this if another device made changes and you want them here now without waiting.

Sign Out — disconnects this device from your account. Your local character data stays on the device — it is not deleted. You can sign back in at any time.

Sync happens automatically in the background — you don't need to press any button during normal play.

Data and Backup — manual tools for saving, exporting, and importing your character data.

Save Data — forces an immediate save of all character data to your browser's local storage. Normally saves happen automatically every few seconds — use this button if you want to be certain.

Export Character — downloads your currently loaded character as a JSON file to your device. Use this as a backup before making big changes, or to move a character to another device without cloud sync. Keep these files somewhere safe.

Import Character — loads a character from a previously exported JSON file. The imported character is added to your character list — it does not overwrite existing characters.

If cloud sync is enabled, exports are still useful as a local backup that doesn't depend on Google or the internet.

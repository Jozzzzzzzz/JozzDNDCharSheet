Notes

Notes — your campaign journal, organised into folders and cards.

Folders — the top-level grid you see here. Each folder is a category. Click any folder to open it and see its cards. Default folders include Quests, NPCs, Locations, Loot, Party, World, Combat, Downtime, Secrets, and Misc.

Cards — inside a folder, each card is a separate note. Click a card to open the full editor and write as much as you like. The card title and body both save automatically as you type.

+ Add Folder — creates a new custom folder. Name it anything (e.g. "Gods & Religion", "Merchant Contacts", "House Rules").

+ Add Card — inside a folder, adds a new note card. Give it a title to find it quickly later.

Reorder — turns on arrows + drag handles so you can move folders (or cards inside a folder) into any order. Click Reorder again to exit.

Delete — a separate mode that reveals a delete button on each folder/card, with multi-step confirmation to prevent accidents. Default folders and their template cards can't be deleted, but you can edit the template content freely. Reorder and Delete are separate — turning one on turns the other off.

← Back returns you to the folder grid from inside a folder. The tab also resets to folder view every time you switch pages.
